# 基于HSV色彩空间的绿地识别算法说明

## 算法思路

本研究采用HSV色彩空间进行绿地识别，核心思路是利用绿色植被在色相（Hue）、饱和度（Saturation）、明度（Value）三个维度上的颜色特征，将卫星影像中的绿地与其他地物区分开来。

## 1. 色彩空间转换

首先将卫星影像从RGB色彩空间转换至HSV色彩空间。HSV模型更接近人眼对颜色的感知方式：H分量表示颜色种类（0°\~180°），S分量表示颜色纯度，V分量表示明暗程度。相比RGB空间中红、绿、蓝三通道相互耦合，HSV空间能更直观地将"绿色"这一颜色属性独立出来。

## 2. 阈值分割

基于绿色植被在HSV空间中的典型分布范围，设定三组阈值进行像素级筛选：

- **色相（H）**：25\~100，对应绿色光谱区间，排除蓝色、黄色等非绿色地物；
- **饱和度（S）**：18\~255，下限排除灰色、白色等低饱和度像素（如建筑屋顶、道路），上限保留所有高饱和度绿色；
- **明度（V）**：8\~220，下限排除极暗区域（阴影），上限排除过亮区域（如反光水面、高亮建筑表面）。

落入该范围的像素被判定为绿地像素。

## 3. 灰色遮罩排除

由于QGIS地图集导出时，分析区域外的区域被填充为RGB(150,150,150)的灰色遮罩，程序首先排除与灰色遮罩颜色相近（容差±15）的像素，仅在有效卫星影像区域内计算绿地率。

## 4. 绿地率计算

$$\text{绿地率} = \frac{\text{绿色像素数} \cap \text{有效像素数}}{\text{有效像素数}} \times 100\%$$

## 优缺点

**优点：**
- 计算速度快，无需训练数据
- 对绿色植被的颜色特征捕捉较为敏感

**局限性：**
- 对深绿色树冠与浅绿色草坪的识别一致性受阈值影响
- 可能将部分绿色屋顶、运动场地等人造绿色表面误判为植被绿地
