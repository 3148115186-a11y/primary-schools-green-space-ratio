# 北京市海淀区小学绿地空间覆盖率分析

基于卫星遥感影像与GIS空间分析，量化评估北京市海淀区约157所小学的校内及周边绿地覆盖率。

## 研究内容

1. **校内绿地率分析**：基于HSV色彩空间的像素级绿地检测，计算学校边界范围内的绿地覆盖率
2. **周边绿地可达性分析**：通过高德地图API，计算每所学校500米范围内的公园绿地数量及步行距离
3. **四象限分类与规划设计**：根据校内/外围绿地率进行四象限分类，提出针对性改造建议

## 技术路线

- **卫星底图**：高德卫星影像（GCJ-02坐标系）
- **截图导出**：QGIS Atlas地图集功能 + 反转多边形遮罩
- **绿地检测**：HSV色彩空间阈值分割（v7参数）
- **可达性分析**：高德周边搜索API + 步行路径规划API
- **学校边界获取**：百度地图API / OpenStreetMap / 高德地图API 多源融合

## 使用方法

### 环境要求

```bash
pip install opencv-python numpy Pillow requests dashscope geopandas
```

### 绿地率分析

1. 准备三组卫星截图（校内/外围/完整），放入对应文件夹
2. 修改 `scripts/core/绿地率计算.py` 或 `scripts/绿地率分析_高德地图.py` 中的路径配置
3. 运行：
```bash
cd scripts/core
python 绿地率计算.py
```

### 公园可达性分析

1. 设置高德地图API Key：
```bash
export AMAP_WEB_KEY=your_api_key_here
```
2. 运行：
```bash
cd scripts
python 学校周边公园分析.py
```

### VLM视觉模型分析

1. 设置DashScope API Key：
```bash
export DASHSCOPE_API_KEY=your_api_key_here
```
2. 运行：
```bash
cd scripts/core
python 分析小学绿地覆盖.py
```

## 分析结果

| 指标 | 数值 |
|------|------|
| 学校总数 | 约157所 |
| 500m内有公园的学校 | 68所（43.6%） |
| 最近公园步行距离 | 58m ~ 3255m |

### 四象限分类

| 分类 | 条件 | 说明 |
|------|------|------|
| 校内高_外围高 | 校内绿地率高 & 外围绿地率高 | 绿地条件优越 |
| 校内高_外围低 | 校内绿地率高 & 外围绿地率低 | 校内优势突出 |
| 校内低_外围高 | 校内绿地率低 & 外围绿地率高 | 周边可借力 |
| 校内低_外围低 | 校内绿地率低 & 外围绿地率低 | 绿地条件薄弱，需重点关注 |
