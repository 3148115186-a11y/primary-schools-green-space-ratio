# 北京市海淀区小学绿地空间覆盖率分析

基于卫星遥感影像与GIS空间分析，量化评估北京市海淀区156所小学的校内及周边绿地覆盖率。

## 研究内容

1. **校内绿地率分析**：基于HSV色彩空间的像素级绿地检测，计算学校边界范围内的绿地覆盖率
2. **周边绿地可达性分析**：通过高德地图API，计算每所学校500米范围内的公园绿地数量及步行距离

## 技术路线

- **卫星底图**：高德卫星影像（GCJ-02坐标系）
- **截图导出**：QGIS Atlas地图集功能 + 反转多边形遮罩
- **绿地检测**：HSV色彩空间阈值分割（v7参数）
- **可达性分析**：高德周边搜索API + 步行路径规划API

## 使用方法

### 环境要求

- Python 3.x
- 依赖：`opencv-python`、`numpy`、`Pillow`

```bash
pip install opencv-python numpy Pillow
```

### 绿地率分析

1. 准备三组卫星截图（校内/外围/完整），放入对应文件夹
2. 修改 `scripts/绿地率分析_高德地图.py` 中的 `CONFIG` 路径配置
3. 运行：

```bash
cd scripts
python 绿地率分析_高德地图.py
```

### 公园可达性分析

1. 设置高德地图API Key：

```bash
export AMAP_API_KEY=your_api_key_here
```

或修改 `scripts/学校周边公园分析.py` 中的 `CONFIG`。

2. 运行：

```bash
cd scripts
python 学校周边公园分析.py
```

## 数据说明

| 数据文件 | 说明 |
|----------|------|
| 绿地率统计结果.csv | 156所学校的校内/外围/总体绿地率 |
| 绿地率分类结果.csv | 四象限分类（校内高/低 × 外围高/低） |
| 绿地率GIS数据.csv | 含WGS84坐标的GIS数据，可直接导入QGIS |
| 学校公园汇总.csv | 每校500m内公园数量、最近公园步行距离和时间 |
| 学校公园明细.csv | 学校-公园配对的详细步行距离和时间 |

## 分析结果

| 指标 | 数值 |
|------|------|
| 学校总数 | 156所 |
| 总体绿地率均值 | 30.5% |
| 总体绿地率中位数 | 29.9% |
| 总体绿地率范围 | 4.7% ~ 70.0% |
| 500m内有公园的学校 | 68所（43.6%） |
| 最近公园步行距离 | 58m ~ 3255m |

### 四象限分类

| 分类 | 条件 | 学校数 |
|------|------|--------|
| 校内高_外围高 | 校内≥30% & 外围≥28% | 48所 |
| 校内高_外围低 | 校内≥30% & 外围<28% | 19所 |
| 校内低_外围高 | 校内<30% & 外围≥28% | 43所 |
| 校内低_外围低 | 校内<30% & 外围<28% | 46所 |
